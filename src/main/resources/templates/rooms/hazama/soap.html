SOAP WEB サービス

SOAP APIレスポンス確認

【コマンド】

①request.xmlの6行目にCountryRepository.javaの国名を入れる

②コマンドプロンプトで以下のコマンドを実行する
curl --header "content-type: text/xml" -d @request.xml http://localhost:8080/ws

③レスポンスが表示される
CountryRepository.javaで設定した内容が返ってきたらOK


【SOAP UI】

①SOAP UIをダウンロード
SoapUI Open Source

②SOAP UIを起動してプロジェクトを作成する
New SOAP Project
Project Name:Web Service Test
Initial WSDL:http://localhost:8080/ws/countries.wsdl

③SOAPリクエストの送信
作成したプロジェクトをダブルクリックし
左側に表示されたSOAPメッセージのnameを
CountryRepository.javaの国名に書き換え
緑色の実行ボタンをクリック

④SOAPレスポンスの確認
右側に表示されたSOAPレスポンスを確認
CountryRepository.javaで設定した内容が返ってきたらOK
